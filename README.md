# Игра-поздравление "С 8 Марта!" 💐

Милая интерактивная игра-поздравление для девушек с Международным женским днём!

## 🎮 Как играть

1. Нажми кнопку "Начать игру"
2. Кликай на появляющиеся цветочки, чтобы собрать букет
3. Собери 10 цветов до того, как время истечёт
4. Каждый цветок приносит очки
5. После победы получишь милое поздравление!

## ✨ Особенности

- **Анимации сердечек** - кликай в любом месте экрана
- **Секретное сердечко** - нажми на сердечко в футере
- **Разные типы цветов** - каждый даёт разное количество очков
- **Поздравления** - случайные тёплые сообщения
- **Адаптивный дизайн** - работает на всех устройствах

## 🎯 Цель игры

Собрать 10 цветов за 60 секунд и получить красивое поздравление с 8 Марта!
